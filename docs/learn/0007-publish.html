Use it to back up your work, move between machines, or hand a finished exhibit to a colleague.

Also the collaboration unit The same .archie.zip is how two people work async: share a copy, each edits, then import the changes back — Archie merges them by version history. No server, no accounts.

Or put it live on the web

To publish a public site, Archie pushes straight to GitHub Pages: give it the owner, repo, branch, and a personal access token (pasted fresh, never stored). It uploads your media, makes the commit, turns Pages on, and hands you a live URL.

For citation & reuse Tick "ship source originals" to include full-resolution files others can cite. Links between exhibits in your library are checked at publish, so nothing points into the void.
