Import notes from a spreadsheet

Hand-drawing every note is slow. If you already have your annotations in a spreadsheet, import them in bulk — one row becomes one Note.

Those are the CSV columns. Grab the template from the import menu so the headers match.

Good to know A row that already has a region (x · y · w · h) lands as a finished note straight away. Tags and reading are optional — fill them and the note arrives already grouped (Step 4).

No coordinates? They wait for you

This is the part people miss: a row with text but no region isn't dropped. It waits in a pending notes drawer until you place it.

Row, no region → → Placed note

Why the extra step Every Note stays anchored to a real region of an Object — so coordinate-free text never becomes a placeless marker floating on the canvas. You import fast, then place with intent.
